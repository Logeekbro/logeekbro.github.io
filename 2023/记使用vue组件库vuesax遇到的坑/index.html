min-height。

实际操作发现动画居然是全屏加载，在几番尝试无果后开始查资料…

发现有一个issue非常符合这种情况：

根据下面老哥的提示又去查看了#633:

第一句话说要在ref后加$el，实测也没用，下面一句话才是重点：

原文：

A second mild annoyance is that unless the button has attribute class=vs-con-loading__container, the load will have a fullscreen behavior, instead of having the load contained within the button itself. This not really explained anywhere.

翻译：

第二个轻微的烦恼是，除非按钮具有属性 class=vs-con-loading__container，否则加载将具有全屏行为，而不是将加载包含在按钮本身中。这在任何地方都没有真正解释。

在尝试给div加上该class后终于成功加载了，至此问题解决。

经过后面的了解发现该功能貌似是其他开源爱好者开发的，并非官方出品的功能，所以可能存在bug。
