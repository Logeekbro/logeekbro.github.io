基本数据类型和包装类

发表于 2023-01-21 分类于 Java

Integer a= 127 与 Integer b = 127相等吗

对于对象引用类型：==比较的是对象的内存地址。

对于基本数据类型：==比较的是值。

如果整型字面量的值在-128到127之间，那么自动装箱时不会new新的Integer对象，而是直接引用常量池中的

Integer对象，超过范围 a1==b1的结果是false
